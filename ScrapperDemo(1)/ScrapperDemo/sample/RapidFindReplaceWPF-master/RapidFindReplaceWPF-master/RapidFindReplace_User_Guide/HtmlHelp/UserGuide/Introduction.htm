Overview

RapidFindReplace WPF provides a find and replace control for WPF applications.  The control offers:

  • 'Find only' or 'find and replace' mode
  • Support for wide range of WPF controls, eg;
    • TextBox
    • RichTextBox
    • ComboBox
    • DataGrid
    • Label
    • ListBox
    • ListView
    • TreeView
    • FlowDocumentPageViewer
    • FlowDocumentScrollViewer
    • Button (disabled by default)
    • IContentHost Controls
    • Non visual collection of Run objects (ie. strings)
    • Controls using any of these as child controls (eg. UserControl, Window etc)
  • Multiple user options similar to those in Office software
    • Find as user types (inline highlighting)
    • Case matching
    • Whole words only
    • Wildcards
    • Regular expressions
    • Prefix/suffix only matching
    • Ignore whitespace
    • Ignore punctuation

Requirements

The components were built to .NET (WPF) 4 (4.x) specification and are compatible with Visual Studio 2010, 2012, 2013. WPF 4 is supported by all Windows OS versions from XP through 8.1.
